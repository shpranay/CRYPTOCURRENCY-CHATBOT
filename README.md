## üí¨ Available Chatbot Commands

You can interact with the Crypto Mentor Chatbot using the following commands:

### 1. Get Real-Time Crypto Price
**Command:**  
`price <coin_name>`  
**Example:**  
`price bitcoin`  
**Description:**  
Returns the current price (in USD) of the specified cryptocurrency.

---

### 2. Estimate Portfolio Value
**Command:**  
`portfolio <quantity coin_name, quantity coin_name, ...>`  
**Example:**  
`portfolio 2 bitcoin, 3 ethereum, 0.5 dogecoin`  
**Description:**  
Estimates and prints the total value of your portfolio based on real-time prices.

---

### 3. Get Crypto Information
**Command:**  
`info <coin_name>`  
**Example:**  
`info solana`  
**Description:**  
Fetches a brief description or introduction of the given cryptocurrency.

---

### 4. View Top 5 Cryptocurrencies
**Command:**  
`top`  
**Description:**  
Displays the top 5 cryptocurrencies by market capitalization, along with their current prices.

---

### 5. Get a Random Crypto Joke
**Command:**  
`joke`  
**Description:**  
Lightens the mood with a random crypto-themed joke.

---

### 6. Help (List All Commands)
**Command:**  
`help`  
**Description:**  
Displays the list of all available commands and their usage.

---

### 7. Exit the Chatbot
**Command:**  
`exit`  
**Description:**  
Closes the chatbot session.
